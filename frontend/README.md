# Frontend Angular Black Coffe

## Levantar frontend

Desde `frontend`:

```powershell
npm install
npm start
```

URL local: `http://localhost:4200`

## Requisito

Backend debe estar levantado primero en `http://localhost:5088`.
El proxy Angular enruta `/api/*` hacia ese host.

## Tema visual

La UI esta en estilo monocromatico (blanco/negro) y mantiene flujo:
- Login / Create Account
- Menu
- Carrito
- Checkout
- Perfil
- Reservas
